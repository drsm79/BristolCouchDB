Bristol CouchDB Meetup

Meeting once a month the Bristol CouchDB Meetup is aimed at anyone interested in using or learning about CouchDB. The meetings are scheduled via Meetup. This page is used to advertise the events to the community and also for scheduling talks. If you would like to speak please submit a pull request with the necessary edits to the page.

First event

Details of our first event are up on Meetup - come along to Cafe Kino on the 30th to chat Couch!
