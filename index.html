Bristol CouchDB Meetup

Meeting once every few months the Bristol CouchDB Meetup is aimed at anyone interested in using or learning about Apache CouchDB. The meetings are scheduled via Meetup. This page is used to advertise the events to the community and also for scheduling talks. If you would like to speak please submit a pull request with the necessary edits to the page.

See our history page for details of talks at previous events.

Second talk night RSVP

Our second talk night will take place on the 13th March at the new UK offices of Cloudant at 25 Hobbs Lane. We'll meet at 7 with the talks starting at 7:30 and each lasting 30 mins (including questions/discussion). We'll have some drinks and nibbles sponsored by Cloudant.

Completing the CouchDB command line jigsaw with jq

Mike Wallace
Cloudant

This talk introduces jq, a command line tool for processing JSON data. The talk will provide an overview of CouchDB tooling on the command line, discuss the key features of jq and conclude with some live demonstrations and examples.

SecondSync: Iterating a data platform

Dan Fairs
SecondSync

The SecondSync data platform has gone through through multiple architecture iterations over the past year. Find out how CouchDB helped us, why we had to leave it behind, and how it may yet become the prodigal database.
